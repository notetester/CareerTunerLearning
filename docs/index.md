## 이 사이트의 학습 흐름

면접에서 말이 막히는 건 보통 "몰라서"가 아니라 **말로 꺼내는 구조가 없어서**입니다.
그래서 이 사이트는 외우는 자료가 아니라 **말하기 훈련 자료**로 만들었습니다. 각 페이지는 같은 순서를 따릅니다.

```text
1. 단어 뜻        →  2. 기본 개념      →  3. 왜 필요한가
4. 내 코드 위치   →  5. 동작 원리      →  6. 면접 답변(3단계)
7. 꼬리질문       →  8. 직접 말해보기  →  ✅ 퀴즈
```
